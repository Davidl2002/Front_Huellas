Información del Sistema de Autenticación de Huellas Dactilares

El sistema utiliza un modelo siamesa mejorado para autenticar a los usuarios basándose en sus huellas dactilares. Este modelo ha sido entrenado con un dataset de huellas y está optimizado para ofrecer alta precisión en la verificación de identidad.

¿Cómo funciona?

El modelo siamesa compara dos huellas dactilares y calcula una distancia entre ellas. Si la distancia es baja (es decir, son similares), se considera que las huellas pertenecen al mismo usuario.

Propósito del Modelo

Este modelo tiene como objetivo autenticar usuarios de manera eficiente, utilizando solo una imagen de la huella dactilar para realizar la autenticación. Se ha optimizado para ofrecer una baja tasa de falsos positivos y negativos.

Uso del Sistema

El sistema se puede integrar en aplicaciones que requieran un alto nivel de seguridad, como el control de acceso o la autenticación en sistemas críticos.
